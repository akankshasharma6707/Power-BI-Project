# Energy Consumption Optimization Dashboard (Power BI)

This Power BI dashboard analyzes energy, water, and gas consumption across store locations to identify optimization opportunities and cost savings scenarios.

## Live Dashboard
https://app.powerbi.com/view?r=eyJrIjoiZmU4YzZjZjgtNzNmYy00ZGI4LTgzNzMtZTBkZjU4ZGIzNTk2IiwidCI6IjQyNzhhNDAyLTFhOWUtNGViOS04NDE0LWZmYjU1YTVmY2YxZSIsImMiOjN9

## Objectives
- Monitor energy usage across stores
- Compare optimization scenarios
- Identify high-consumption equipment
- Evaluate cost and emission impact

## Key Insights
- Optimization scenario reduces electricity cost ~29%
- Water cost reduced ~48%
- HVAC & extraction highest energy consumers
- Equipment-level analysis enabled targeted savings

## Features
- Scenario comparison (Current vs Savings vs Must-Have)
- Multi-page navigation (Electricity, Emission, Water, Gas)
- Store & equipment drilldowns
- Interactive tooltips
- Geographic store mapping
